## SourceV1

**SourceV1** is a Discord bot built to streamline the sharing and organisation of academic assessment resources within a server. It provides a structured, channel-based system where users can submit and retrieve assessment materials across multiple subjects, improving collaboration and reducing fragmentation of resources.

As new assessment tasks are introduced, SourceV1 supports the creation of task-specific channels to ensure submissions remain organised, discoverable, and relevant. This prevents resource clutter and enables users to quickly locate materials associated with a particular task or subject.

## Key Features

- **Resource Submission**
  - Users can upload and share assessment resources directly within Discord.
  - Supports multiple subjects and task types.

- **Task-Specific Organisation**
  - Dedicated channels for individual assessment tasks.
  - Ensures resources are grouped logically and remain easy to navigate.

- **Collaborative Access**
  - Community-driven sharing model encourages contribution and reuse.
  - Enables fast access to peer-submitted materials.

- **Scalable Server Design**
  - Designed to operate cleanly in larger servers without resource duplication.
  - Channel-based architecture prevents content overlap.

## Technology Stack

- **Language:** Python  
- **Framework:** Pycord  
- **Platform:** Discord  

## Intended Use

SourceV1 is intended for educational Discord servers that require centralised storage of assessment resources, clear separation of content by subject and task, and a Discord-native alternative to external file repositories.
